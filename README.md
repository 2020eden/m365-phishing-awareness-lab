# Microsoft 365 Phishing Awareness Platform

A phishing awareness platform built using open-source technologies and Microsoft 365.

## Features

- Linux hosted GoPhish
- Microsoft 365 SMTP
- HTML Landing Pages
- Microsoft Entra ID
- DNS configuration
- Security Awareness Campaigns
- Reporting
- PowerShell Automation

## Technologies

- GoPhish
- Ubuntu
- Microsoft 365
- Exchange Online
- Microsoft Entra ID
- HTML
- CSS
- JavaScript
- Python
- PowerShell
